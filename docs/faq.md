# 常见问题

## FORGE Lite 是 PRD 模板吗？

不是。

FORGE Lite 是产品经理工作流 Agent 框架。PRD 生成只是其中一个模块。

## 它包含产品知识吗？

不包含。

使用者需要自己配置产品背景、知识库、代码库和业务 Rubric。

## 没有代码库也能用吗？

可以。

如果任务不依赖代码事实，可以跳过逆向工程模块。

## 非 SaaS 产品能用吗？

可以。

这套框架是通用的，但每个团队都需要配置自己的对象模型、流程、角色、Rubric 和模板。

## 每次都要跑完整流程吗？

不需要。

FORGE Lite 的原则是按需激活模块。简单任务可以只跑一个模块。

## 团队可以加自己的私有 Rubric 吗？

可以，而且应该加。

公开仓库只提供通用 Rubric。具体业务逻辑应该放在私有配置里。

## 为什么文件名是英文？

为了以后开源到 GitHub 时更通用，也方便工具识别。

但正文可以先中文，等框架稳定后再补英文版。
